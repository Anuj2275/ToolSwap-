# ToolSwap: Peer-to-Peer Tool Sharing Platform

ToolSwap is a full-stack application designed to facilitate the lending and borrowing of tools within a community, built with a robust Spring Boot backend and a modern React frontend.

* **Goal:** To create a community-driven platform where users can safely list, discover, and book tools, minimizing waste and maximizing resource sharing.
* **Key Features:** User Registration & Authentication (JWT), Tool Listing (CRUD operations), Image Uploads (Cloudinary), Booking/Lending Management, Email Notifications.
